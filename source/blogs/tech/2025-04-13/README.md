# 2025Q2的健身计划

### 总体目标：Q2 end的时候，体重可以降到80kg以内。

### 日常健身计划：

#### 手和胸
    - 上胸斜推： 凳子2格倾斜，杠铃10kg*2，15次*4组
    - 手臂哑铃弯举： 20kg， 15次*4组
    - 自由哑铃上推，凳子斜45度： 7.5lb， 15次*4组，手臂垂直于地面
    - 三头下压： 10kg， 15次*4组
    - 平板支撑，1min*4次
    - 有氧 45min， 同时听英语听力
        - b站，youTube口语精选   

#### 肩和背
    - 双手通过肩背拉重物
    - 单杠下拉
    - 肩膀斜上推
    - 水平直拉
    - 自由哑铃上推，凳子90度   

#### 腿和其他
    - 举重物站起和蹲起
    - 卧斜蹬腿
    - 大腿器械
    - 扩大腿外展
    - 小腿反举
