#### AI学习
- 产品形式： 
    - 网站
    - app
    - 微信小程序/公众号
    - 第三方平台仅提供内容：b站，知乎等
    - 自己录视频，制作课件
- 痛点：
    - 各种资料太多了，质量层次不齐，需要浪费大量时间去筛选
    - 没有成名的产品形成品牌
    - 需要高屋建瓴的把整片森林的全貌展现出来， 没有成长路径
    - 没有结合各个leading 公司的最新材料
